## Getting Started

```sh
mkdir project-name
cd project-name
```
```sh
forge init --template https://github.com/rohansanjay/ft
forge install
```

## Development

**Building**
```bash
forge build
```

**Testing**
```bash
forge test
```

**Updating Gas Snapshots**
```bash
forge snapshot
```
